## Portfolio

Personal portfolio website built with Next.js, TypeScript, and Tailwind CSS.

## Overview

This portfolio presents my profile, skills, projects, services, resume, and contact details in a single-page layout. It includes smooth scroll animations, responsive navigation.

## Features

- Responsive landing page with a mobile-friendly navbar
- Hero section with animated intro content
- Skills, projects, services, resume, and contact sections
- Scroll animations powered by AOS

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- AOS for scroll animations
- React Icons
- tsParticles
- react-parallax-tilt
- typewriter-effect

## Project Structure

- `app/` - App Router pages and global styles
- `components/` - Portfolio sections and shared UI
- `constant/` - Navigation links and shared constants
- `public/` - Static assets and images

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Available Scripts

- `npm run dev` - start the development server
- `npm run build` - create a production build
